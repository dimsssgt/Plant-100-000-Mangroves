# Plant-100-000-Mangroves
tugas week1 

Website kampanye penanaman **100.000 pohon mangrove** dengan tampilan modern dan nuansa hijau alami.  
Proyek ini dibuat sebagai bagian dari tugas desain web menggunakan **HTML dan CSS** tanpa JavaScript,  
dengan fokus pada tata letak yang responsif, warna ramah lingkungan, dan tipografi elegan.

## Fitur Utama
- Tema hijau gradasi yang konsisten dari atas hingga bawah halaman  
- Section utama: *Home*, *About*, *Impact*, *How It Works*, *Donate*, dan *Contact*  
- Tata letak gambar dan teks yang seimbang (tanpa mengubah struktur HTML utama)  
- Efek hover lembut pada tombol dan elemen interaktif  
- Desain minimalis, bersih, dan mudah dibaca di perangkat kecil
